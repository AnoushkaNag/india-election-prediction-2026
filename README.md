# India Election Prediction 2026 🇮🇳

Hybrid Machine Learning + Rule-Based system to predict constituency-level election outcomes.

## 📌 Objective
Predict winning candidates across constituencies using historical data, candidate profiles, and voting patterns.

## ⚙️ Approach
- Rule-based heuristics (incumbency, margin, turnout)
- Machine Learning model (classification)
- Hybrid decision system combining both

## 📂 Project Structure
- `data/` → raw and cleaned datasets  
- `notebooks/` → exploratory analysis  
- `src/` → core pipeline (rules + ML)  
- `outputs/` → final predictions
